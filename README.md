# Overview

## Next.js Core Training

is a developer-focused training repository designed to build strong foundations in modern Next.js application development.

The repository focuses on:

- Understanding Next.js architecture
- Building reusable React components
- Learning App Router concepts
- Exploring server and client components
- Implementing modern rendering strategies
- Working with APIs and data fetching
- Following scalable project structures
- Applying production-ready coding practices

This repository contains examples, experiments, notes, and practical exercises for improving Next.js development skills.
